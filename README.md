# SimplePOS-SpringREST-AngularJS

Program sederhana point of sale.
terdapat 3 master :
- Barang
- Customer
- Supplier

dan 2 transaksi :
- Penjualan
- Pembelian

Requirements :
- Netbeans >= 8.0.1
- Mysql

How-to :
1. Import .sql file dari folder Database
2. Running menggunakan Apache tomcat server.
3. Enjoy!
