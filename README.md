# SimplePOS-SpringREST-AngularJS

Program sederhana point of sale. Berbasis java spring rest dan angularjs.

terdapat 3 master :
- Barang
- Customer
- Supplier

dan 2 transaksi :
- Penjualan
- Pembelian

#### Requirements :
- Netbeans >= 8.0.2
- Mysql

#### How-to :
1. Import .sql file dari folder Database
2. Running menggunakan Apache tomcat server.
3. Enjoy!
